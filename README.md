# Saudi Forecasting Platform

An open-source, multi-horizon probabilistic forecasting platform for Saudi/Gulf electricity demand and PV generation. Built with ensemble models, conformal prediction intervals, hierarchical reconciliation, and a bilingual Arabic/English LangGraph agent.

## Overview

This platform produces 1-hour, 6-hour, 24-hour, and 7-day forecasts at national, regional, and plant levels. Every forecast is accompanied by a calibrated prediction interval, a causal-vs-correlational driver decomposition, and a natural-language explanation generated by a LangGraph agent grounded in Saudi energy literature via retrieval-augmented generation (RAG).

## Key Features

- **Multi-horizon forecasting** across 1h / 6h / 24h / 7-day horizons
- **Ensemble modelling** (LightGBM + LSTM + Prophet, ridge-stacked)
- **Conformal prediction intervals** with distribution-free coverage guarantees
- **Causal analysis** separating Granger-causal drivers from SHAP-correlational attribution
- **Hierarchical reconciliation** (bottom-up, top-down, MinT) across plant -> region -> national
- **Drift detection** (PSI) with automated retraining triggers
- **Bilingual LangGraph agent** (Arabic/English) with grounded RAG retrieval

## Tech Stack

Python, FastAPI, LightGBM, MAPIE, LangGraph, ChromaDB, Anthropic Claude, MLflow, Docker, GitHub Actions, AWS.

## Inspired By

This project is an open-source parallel to two publicly disclosed Saudi initiatives:

1. The **Aramco x ACWA Power AI photovoltaic energy forecasting agreement**, announced at ACWA Power Innovation Days, Riyadh, February 2025 (source: SolarQuarter, 11 Feb 2025).

2. **ACWA Power's Monitoring and Prediction Centre (MPC)**, recognised by the International Telecommunication Union (ITU) during COP30 (Brazil, 13 November 2025) as a Saudi flagship AI-for-climate initiative. The MPC is projected to support annual emissions reductions of approximately 588,717 tonnes of CO2 by 2030 through AI-driven forecasting and optimisation across ACWA Power's fleet (sources: CST.gov.sa statement N2025111301, 13 Nov 2025; ACWA Power official site).

This repository implements the same problem class -- multi-horizon probabilistic load and PV forecasting on Saudi-relevant data -- entirely on public datasets, so any reviewer can clone, audit, and extend it.
